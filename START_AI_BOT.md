# 🤖 Запуск NutriBuddy Bot с реальным AI

## 📋 Предварительная настройка:

### 1. Установите переменные окружения Cloudflare

**Windows (PowerShell):**
```powershell
$env:CLOUDFLARE_ACCOUNT_ID = "ваш_account_id_от_cloudflare"
$env:CLOUDFLARE_API_TOKEN = "ваш_api_token_от_cloudflare"
```

**Windows (CMD):**
```cmd
set CLOUDFLARE_ACCOUNT_ID=ваш_account_id_от_cloudflare
set CLOUDFLARE_API_TOKEN=ваш_api_token_от_cloudflare
```

**Linux/Mac:**
```bash
export CLOUDFLARE_ACCOUNT_ID="ваш_account_id_от_cloudflare"
export CLOUDFLARE_API_TOKEN="ваш_api_token_от_cloudflare"
```

### 2. Проверьте настройку OpenWeatherMap (для климатических рекомендаций)
```powershell
$env:OPENWEATHERMAP_API_KEY = "ваш_api_key"
```

## 🚀 Запуск бота:

### 1. Проверьте работу AI:
```bash
python scripts/test_enhanced_ai.py
```

Должно появиться:
```
🧠 Hermes: Cloudflare AI настроен для реальной работы
🧠 Hermes: успешно сгенерирована аналитика
```

### 2. Запустите основного бота:
```bash
python bot.py
```

## 🎯 Что делает AI в боте:

### 🧠 Enhanced AI Parser (100% AI):
- **Распознает еду**: "съел 200г курицы с гречкой"
- **Анализирует воду**: "выпил большую кружку чая" → 300мл
- **Определяет активность**: "интенсивная тренировка 45 минут"
- **Классифицирует намерения**: понимает что хочет пользователь

### 🌤️ Climate Manager (100% AI):
- **Анализ погоды**: получает данные и адаптирует рекомендации
- **Климатические советы**: "увеличить калории в холодную погоду"
- **Гибкие рекомендации**: AI анализирует температуру и влажность

### 🥗 Nutrition Calculator (100% AI):
- **Рассчитывает КБЖУ**: персонализированные нормы
- **Анализ прогресса**: AI определяет тенденции
- **Умные рекомендации**: адаптирует под цели пользователя

### 🔄 AI Integration Manager (мозг бота):
- **Маршрутизация запросов**: AI решает какой компонент вызвать
- **Обогащение результатов**: AI улучшает ответы
- **Контекстная обработка**: помнит историю диалога
