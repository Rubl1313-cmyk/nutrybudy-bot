# 📋 Обновления NutriBuddy Bot - Март 2026

## ✅ Что изменено:

### 🌍 **1. Автоматический часовой пояс из города**
- Больше не нужно выбирать часовой пояс вручную
- Введите город (например: "Москва", "Новосибирск", "Екатеринбург")
- Часовой пояс определится автоматически
- Поддерживаются все основные города России и СНГ

### 📊 **2. Умный расчет шагов**
- Больше нет жесткой цели 10000 шагов для всех
- Цель рассчитывается персонально на основе:
  - Возраста и пола
  - Уровня активности
  - Цели (похудение/набор/поддержание)
- Примеры:
  - Мужчина, 30 лет, умеренная активность, похудение: **13,200 шагов**
  - Женщина, 45 лет, легкая активность, поддержание: **7,600 шагов**

### 📏 **3. Расчет состава тела из обхватов**
- Удалены поля мышечной/жировой массы (нужно было измерять)
- Теперь состав тела рассчитывается автоматически из обхватов:
  - Шея, талия, бедра (обязательно)
  - Бицепс, предплечье, грудь (дополнительно)
- Расчет по формуле ВМС (US Navy) - золотой стандарт
- Точность: ±2-3%

### 📅 **4. Еженедельные напоминания**
- Автоматические напоминания раз в неделю
- Напоминают обновить вес и обхваты
- Показывают текущие цели и прогресс
- Мотивируют продолжать отслеживать данные

### 🗣️ **5. Упрощение языков**
- Удалена поддержка множества языков
- Остался только русский язык
- Упрощен код и уменьшен размер бота

### ⚙️ **6. Исправлены настройки**
- Удалены неработающие кнопки
- Оставлены только рабочие функции:
  - 👤 Мой профиль
  - 🎯 Мои цели  
  - 📊 Статистика
  - 🏠 Главное меню

## 🚀 Как использовать новые функции:

### 🌍 **Настройка часового пояса:**
1. Настройте профиль: `/set_profile`
2. Введите город (например: "Москва")
3. Часовой пояс определится автоматически
4. Готово! Напоминания будут приходить вовремя

### 📊 **Расчет шагов:**
1. Настройте профиль с вашими данными
2. Цель по шагам рассчитается автоматически
3. Отслеживайте прогресс: "5000 шагов", "12000 шагов"

### 📏 **Измерение состава тела:**
1. Измерьте обхваты сантиметровой лентой:
   - Шея - на уровне щитовидного хряща
   - Талия - на уровне пупка
   - Бедра - самая широкая часть
2. Введите данные в профиле
3. Получите расчет жира/мышц/воды

### 📅 **Еженедельные напоминания:**
1. Включите напоминания в профиле
2. Раз в неделю бот будет напоминать обновить данные
3. Обновляйте вес и обхваты для точного прогресса
